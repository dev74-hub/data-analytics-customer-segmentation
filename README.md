# Customer Segmentation using K-Means

## Project Overview
This project performs Exploratory Data Analysis (EDA) and customer segmentation using K-Means clustering on the iFood marketing dataset.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## How to Run
1. Install dependencies:
   pip install -r requirements.txt
2. Download the dataset from Kaggle (Marketing Campaign Dataset).
3. Place `ifood_df.csv` in the same folder as the Python file.
4. Run the script.

## Dataset
Marketing Campaign Dataset (Kaggle)
